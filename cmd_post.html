post

The post command generates a new post Markdown file in the source/_posts directory of your Octopress site. You specify the name of your post as the argument for this command.

This command automatically adds SEO meta fields (description & keywords) and a field to that allows you to indicate that the post is a draft (published) to the default Octopress post YAML frontmatter. Octopus automatically includes the date that the Markdown file is created in the Markdown filename string. This date is carried forward to the final static HTML file name when you use the generate or publish commands.

Usage

oc post <post name>

Arguments

Argument Description
post name the title of your post

Options

Option Description
--octo use the default Octopress/Jekyll post generator

Examples

Default Octopus Post Generator

To create the Markdown filename [DATE]-this-is-my-post-title.markdown which will generate the HTML post [DATE]-this-is-my-post-title.html on your site, use the following command:

oc post "This is My Post Title"

The Markdown file creation date is automatically added to the post filename string. This date is also used in the filename string for the static HTML file that is generated from your Markdown.

The character cases that you use in the post name argument are carried forward to the title YAML frontmatter in your Markdown file. This will be the <title> tag in your deployed static HTML file. Octopus automatically converts all characters to lower case and replaces spaces with dashes for the Markdown filename.

Note this filename format if you use the find command or any of the commands that allow you to work with files by Markdown filename substring.

Octopress Generator

If you would prefer to use the default Octopress new post generator, use the --octo switch with this syntax:

oc post --octo "This is My Post Title"

This switch eliminates the additional YAML frontmatter information that OC adds to your post. Your new post Markdown is generated as if you entered the following Octopress/Jekyll command:

rake new_post["This is My Post Title"]
