list

The list command displays a list of all post Markdown files in the path /source/_posts. You can use this command from any directory in your main Octopress site directory.

Posts are listed in a paginated format. Press the space bar to go to the next page. Enter q to exit back to the main terminal.

Usage

oc list

Arguments

None

Options

None
