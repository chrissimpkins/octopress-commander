preview

The preview command starts a local server in the public directory of your site so that you can view changes as you edit your content and site files.

The command provides the option to specify the port on which you will run the server. The default port is 8000 which is accessible at http://localhost:8000 in your browser.

Usage

oc preview <port>

Arguments

Argument Description
port (optional) define the port for your server

Options

None
