doctor

A utility command that checks for installed application dependencies. It scans your system for both Octopress/Jekyll and Octopus dependencies. The report is displayed in your terminal.

Note that these tests include some command-specific dependencies that may not be relevant to you if you do not use the command. Fear not if it reports a missing dependency and the application is working just fine.

Usage

oc doctor

Arguments

None

Options

None
