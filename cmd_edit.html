edit

The edit command launches a post Markdown file in the user-defined default text editor. You define the default editor by creating the $OCEDITOR variable in your shell startup file.

This command is automatically run as part of the workflow sequence when you use the write command.

Usage

oc edit <post filename substring>

Arguments

Argument Description
post filename substring a substring in the Markdown filename

Options

None
