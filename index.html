Octopress Commander

A tool that simplifies the management of your Octopress powered site.

View project onGitHub

Octopress Site Management... Simplified

Example Workflow

Create a New Post Markdown File

oc post "A Really Cool Post"

Start a Local Server to Edit Your Post

oc preview

Watch for Changes in Your Octopress Directories as You Edit

oc watch

Publish Your Final Draft (GitHub or rsync)

oc publish

Publish Your Final Draft (Heroku)

oc publish --heroku
